This is a Test Java - Maven based project developed using Eclipse IDE. You may import it to your IDE as a maven project and run JUnit test cases.

Environment details: 
JDK 1.8 
Log4J for logging
JUnit for testing
Maven for dependency management

Steps: 
1. Import as Maven project
2. Set project to use JDK and JRE 1.8

Directory Structure
source : src/main/java/
test : src/test/java

I have provided implementation for Payroll Range Container using two approaches: 
1. NavigableMap based approach using a TreeMap found under : com.workday.index.payroll
2. Collections Query Engine - NoSQL index and search based approach found under : com.workday.queryengine.payroll
	https://github.com/npgall/cqengine
